# Голубкова Вероника ИСП251

## Инструкция по использованию приложения для пользователей.

**Название приложения:** «Список покупок»  
**Автор ФИО группа:** Голубкова Вероника Андреевна ИСП251  
**Платформа:** Android  
**Язык:** Java  
**База данных:** Room  

Приложение помогает быстро составлять список покупок, искать и сохранять любимые товары. Все данные хранятся только на Вашем телефоне. Интернет не требуется.

### 1. Главный экран  
На нём Вас встречает название приложения. Чуть ниже располагаются актуальные дата/время и три кнопки:  
• Список покупок — Ваш активный список.  
• Избранное — товары, которые Вы покупаете часто.  
• Поиск товаров — окно для добавления и поиска товаров.  

### 2. «Список покупок»  
Как добавить товар:  
• Нажмите «Поиск товаров», затем найдите нужный и кликните на плюс («Добавить в список»).  
• Очистить список (кнопка вверху экрана) — удалит все товары из списка.  

### 3. «Избранное»  
Здесь хранятся Ваши часто покупаемые товары.  
• Вы можете добавить сюда товар из списка покупок или поиска, нажав на сердечко в карточке товара.  
• Откройте «Избранное», затем выберите товар и нажмите на знак плюса («Добавить в список»), чтобы быстро перенести его в список покупок.  

### 4. «Поиск товаров»  
• При нажатии на кнопку с плюсом внизу экрана можно создать новый товар.  
• Введите название товара в строку поиска.  
• Используйте фильтр (кнопка под строкой поиска) для сортировки по магазину, категории, минимальной и максимальной цене.  
• Найденный товар можно: добавить в список, в избранное, отредактировать или удалить из приложения вовсе.  

### 5. Редактирование товара  
При создании товара или после нажатия на знак карандашика («Редактировать») в карточке товара открывается данное окно.  
• В экране «Детали товара» меняйте: название, магазин, категорию, цену.  
• Изменения сохраняются после нажатия на кнопку «Сохранить» ниже.  
• Вы также можете нажать на кнопку «Отмена», если вдруг передумали менять данные товара.  

### 6. Карточка товара  
Мини окошко, где кратко видны все данные о товаре. С помощью него Вы можете:  
• +/- «Добавить в список»/«Удалить из списка» — добавить или удалить товар из списка покупок.  
• ♥ «Добавить в избранное» — сохранить в раздел «Избранное» для быстрых покупок в будущем. Эта же кнопка позволяет убрать товар из избранного.  
• ✏️ «Редактировать» — изменить данные о товаре.  
• 🗑️ «Удалить» — удалить товар из приложения.
